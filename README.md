# 🚀 Project Management System

A modern project management platform designed to help teams efficiently plan, organize, track, and deliver projects. It provides task management, team collaboration, project timelines, reporting, and real-time progress tracking.

---

### 📁 Project Management
- Create, update, archive, and delete projects
- Project status tracking
- Priority levels
- Due dates
- Project descriptions
- Project members

### ✅ Task Management
- Create and assign tasks
- Task priorities
- Due dates
- Labels & tags
- Task comments
- File attachments
- Task history
- Checklist support

### 👥 Team Management

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Project-Manegment/
```

## Navigate

```bash
cd project-management
```

## Install Dependencies

```bash
npm install
```

or

```bash
pnpm install
```

## Configure Environment

Create a `.env` file

```env
DATABASE_URL=

JWT_SECRET=

NEXT_PUBLIC_APP_URL=

CLOUDINARY_NAME=

CLOUDINARY_KEY=

CLOUDINARY_SECRET=
```

## Start Development Server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
